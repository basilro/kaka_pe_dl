# kaka_pe_dl

카카오페이지(page.kakao.com) 웹툰·웹소설의 무료/기다무 회차를 자동으로 내려받는 FlaskFarm 플러그인.

## 주요 기능

- **설정** — 쿠키(로그인) 주입, 다운로드 경로, 기다무/소장권 사용 여부, 출력 포맷,
  프록시, 압축, 알림 웹훅 등을 설정한다. 웹툰 목록(`titles`)과 소설 목록(`titles_novel`)을
  분리해 관리한다.
- **연재 목록** — 카카오페이지 요일별 연재작 전체를 한 화면에서 불러와, **웹툰/소설 분류 토글**과
  요일 필터·검색으로 원하는 작품을 골라 **체크 한 번으로 다운로드 목록에 등록**한다.
  작품 코드를 일일이 찾아 입력할 필요가 없다. 웹툰은 웹툰 목록에, 소설은 소설 목록에
  자동으로 분류되어 등록되며, 이미 등록된 작품은 중복되지 않는다.
- **수동 다운로드** — URL 또는 작품 코드로 특정 작품을 즉시 내려받는다.
- **진행 상황** — 현재 다운로드 진행 상태를 실시간으로 확인한다.
- **다운로드 이력** — 처리된 작품/회차 기록을 조회한다. 소설은 EPUB으로 묶을 수 있다.

## 동작 방식

- 등록된 작품의 무료·기다무·소장 회차를 스케줄에 따라 자동 수집한다. 웹툰은 이미지,
  소설은 텍스트(EPUB)로 저장한다.
- 카카오페이지 BFF API를 사용하며, 로그인 쿠키로 사용자별 보유 티켓/대기무료 상태를 반영한다.
- 연재 목록은 공개 landing API(`view/v2/landing/dayofweek`)에서 가져오므로 로그인이 필요 없다.
