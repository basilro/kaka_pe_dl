📖 카카오페이지 다운로드 — 사용 안내

빠른 시작

  1. 쿠키 주입 — 카카오페이지 로그인 → Cookie-Editor로 JSON export → 설정 > 일반 > 카카오 쿠키에 붙여넣기.
  2. 경로/작품 등록 — 다운로드 경로 지정 → 체크할 웹툰 / 체크할 소설에 한 줄에 하나씩(제목·URL·숫자 ID 모두 인식).
  3. 실행 — 설정 저장 → 쿠키 검증 → 즉시 실행 또는 자동 탭에서 스케줄러 등록.

탭 안내

탭용도
설정 > 일반 카카오 쿠키, 다운로드 경로, 체크할 웹툰/소설, 이용권 옵션, 회차 ZIP 압축, 프록시.
설정 > 알림 Discord/Slack 웹훅. 쿠키 만료 · 웹툰 완료 · 소설 완료 · 완결 자동 제거 알림을 각각 켜고 끔.
설정 > 자동 스케줄러. Interval(분) 또는 Cron 형식. SJVA 시작 시 자동 실행도 옵션.
연재 목록 카카오페이지 연재작을 조회 · 선택해서 체크할 웹툰/소설(스케줄)에 바로 등록. 소설은 장르(판타지 · 현판 · 로맨스 · 로판 · 무협 · BL) 필터 + 랭킹 통합(요일 미배정 작품 포함). 이미 등록된 작품은 표시됨.
수동 다운로드 작품/회차 URL(또는 ID)을 분석해 회차를 선택해서 즉석 다운. 이미 받은 회차는 비활성.
진행 상황 자동/수동 다운로드의 작품 · 회차 · 페이지 진행률 실시간 표시.
다운로드 이력 회차 단위 다운 이력. 상태 · 제목 검색 · 웹툰/소설 종류 필터, 선택 삭제 지원.

저장 구조

다운로드경로/
├─ webtoon/작품제목/
│   ├─ info.xml · cover.jpg
│   ├─ 0001_회차제목/        (압축 Off)
│   └─ 0002_회차제목.zip      (압축 On — 폴더 대신 zip)
└─ novel/작품제목/0001_회차제목.txt

도구 버튼 (설정 탭 상단)

버튼동작
즉시 실행스케줄 무시하고 자동 다운로드 1회 실행.
쿠키 검증현재 쿠키 로그인 상태 확인.
메타 동기화웹툰/소설 폴더를 스캔해 info.xml/cover.jpg 누락분 생성.
압축 처리모든 웹툰 회차 폴더를 zip으로 묶고 원본 삭제(소설·이미 zip인 회차는 제외).
epub 만들기모든 소설 시리즈의 .txt를 epub으로 변환. 소설 EPUB 회차별 개별 생성 설정에 따라 합본(기본) 또는 회차별 개별 파일로 만듦.
이력 초기화DB의 다운로드 이력만 비움(파일은 그대로).

꼭 알아둘 점

  • 이용권: 기다무 사용 + 충전돼야 잠금 회차를 받습니다. 일반(보유) 대여권 사용은 보유 잔량까지 사용. 둘 다 Off면 잠금은 스킵하고 무료/소장만 받습니다. 대여권을 구매하지는 않습니다.
  • 웹툰 압축(ZIP)을 켜면, 이미 디스크에 같은 zip이 있는 회차는 재다운로드하지 않고 이력에만 등록합니다 → 대여권 재차감 방지. (소설은 디스크 인식 안 함)
  • 작품당 신규 다운 상한: 한 번 실행에 작품당 최대 10회차까지만 새로 받습니다(무료/보유 · 잠금 회차 합산). 나머지는 다음 실행에서 이어서 받습니다.
  • 쿠키 만료는 보통 30일. 만료 감지 시 웹훅으로 1회 알림. 재로그인 → Cookie-Editor 재-export → 붙여넣기.
  • 완결+전회차 완료로 인식된 작품은 자동으로 체크 목록에서 제거됩니다.
  • 같은 제목이 웹툰/소설 양쪽에 있어도 등록한 칸의 종류로 정확히 매칭됩니다(체크할 웹툰/체크할 소설).
  • 소설 EPUB: 소설을 받으면 자동으로 epub을 만듭니다. 소설 EPUB 회차별 개별 생성이 Off(기본)면 시리즈 전체를 하나의 합본 epub으로, On이면 회차(.txt)마다 개별 epub으로 만듭니다. 자동 다운로드와 epub 만들기 버튼 모두 이 설정을 따릅니다.
이 플러그인은 본인이 정당하게 접근 권한을 가진 콘텐츠를 개인 보관 목적으로 내려받는 용도입니다. 카카오 이용약관과 저작권법을 준수해 사용하세요.
